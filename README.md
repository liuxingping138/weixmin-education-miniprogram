# 智慧教育小程序

一个专为在线教育老师设计的学生管理工具，帮助老师更好地管理和了解学生。

## 🎯 项目概述

智慧教育小程序是一个轻量级的学生管理系统，核心功能包括：

- **学生管理**: 学生基础信息的增删改查
- **标签系统**: 为学生打标签，便于分类管理  
- **数据概览**: 学生数量统计和标签分布展示

## 🚀 快速开始

### 环境要求
- 微信开发者工具
- Node.js 16+

1. 克隆项目到本地
2. 使用微信开发者工具打开 `frontend` 目录
