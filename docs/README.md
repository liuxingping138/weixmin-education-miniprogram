## 🎯 项目概述

智慧教育小程序是一个专为在线教育老师设计的学生管理工具，核心功能包括：

- **学生管理**: 学生基础信息的增删改查
- **标签系统**: 为学生打标签，便于分类管理
- **数据概览**: 学生数量统计和标签分布展示
